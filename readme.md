# User Management React App

---

A simple React-based CRUD-style application for managing users.
It allows you to view, add, sort, and refresh user data fetched from a local REST API.

## Features

Search Users — Filter users with a search bar

User List — Display all users retrieved from a JSON store

Add New Users — Open a modal form to create new users

Refresh Data — Dynamically refresh the user list after updates

Sort by Creation Date — Sort users by the latest created entry

Pagination Support — Paginate user lists for better navigation
